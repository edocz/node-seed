# Boilerplate for a Koa2 Redux React app with Webpack3, Jest, SASS, Babel, ESLint...

## Instructions

1. **Clone repo**

    git clone https://github.com/edocz/node-seed.git

2. **Install dependencies**

    npm install

3. **Run the webserver**

    npm run webserver

    Webserver should be running at http://localhost:3000/

4. **Make build**

   npm run build

5. **Run development server**

   npm run dev

   Development server should be running at http://localhost:8080/

## Scripts
- **npm run webserver**

     Starts Koa server

- **npm run test**

     Runs unit tests

- **npm run build**

     Bundles the application

- **npm run dev**

     Starts webpack development server

- **npm run lint**

     Runs the linter

- **npm run deploy**

     Creates the production ready files

- **npm run clean**

    Removes the bundled code and the production ready files
